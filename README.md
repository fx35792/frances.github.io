### 安装说明

1、fork库到自己的github <br/>
2、修改名字为：`username.github.io`<br/>
3、clone库到本地，参考`_posts`中的目录结构自己创建适合自己的文章目录结构<br/>
4、修改CNAME(评论)，或者删掉这个文件，使用默认域名<br/>
5、修改`_config.yml`配置项<br/>
6、It's done!<br/>
